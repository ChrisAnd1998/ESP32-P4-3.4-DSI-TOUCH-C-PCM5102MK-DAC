# ESP32-P4-3.4-DSI-TOUCH-C-PCM5102MK-DAC
A simple project to create a music/radio player using a ESP32-P4 Module DEV KIT, PCM5102MK DAC Audio Decoder Module V2.0 - I2S and Waveshare 3.4inch DSI Round Capacitive Touch Display, 800×800 Resolution, IPS Display Panel, 10-Point Touch (3.4-DSI-TOUCH-C)
<br><br>
## Notes
// Tools -> PSRAM -> Enabled<br>
// Tools -> Partition Scheme -> Huge App<br>
<br>
// PCM5102MK<br>
// BCK -> 27<br>
// DATA -> 25<br>
// LRCK -> 26<br>
// GND -> GND<br>
// VCC -> 5V<br>
